STYDL Library

A modular Rust library for high-fidelity spectral audio synthesis.

STYDL is the heart of the AXIS resampler. It is designed as a standalone, modular library for spectral analysis and synthesis. It implements a modern Harmonic plus Noise Model (HNM) that provides smooth, crystal-clear vocals even at extreme pitch and time scales.

Dual-Stream Synthesis Engine

The core synthesis engine implements a hybrid Dual-Stream approach that separates periodic and stochastic components for maximum naturalness:

  • Sinusoidal Stream: High-fidelity sum-of-sines with independent phase accumulators to ensure coherence.
  • Subtractive Noise: White noise shaped by the spectral envelope, providing artifact-free consonants.
  • Stochastic Jitter: RNG-based phase micro-variations to prevent "frozen" or "metallic" textures.

HMM Pitch & Voicing Analysis

STYDL now includes a robust Hidden Markov Model (HMM) layer that performs global optimization on the pitch contour:

  • Viterbi Decisions: Solves the Voiced/Unvoiced state sequence to prevent "humming" on consonants.
  • Multi-pass Smoothing: Combines HMM interpolation with Median filtering to eliminate pitch detection spikes.

Integration Example

STYDL can be integrated into any Rust project needing spectral processing:

// Initialize the engine
let mut engine = StydlEngine::new(sample_rate, fft_size);

// Synthesize from spectral envelopes
let audio = engine.synthesize(&f0_vec, &spec_vec, &ap_vec);

Portability

Unlike many other vocoder implementations, STYDL is written in 100% safe Rust (where possible) and has zero dependencies on legacy C/C++ libraries like WORLD. This makes it trivial to compile for Windows, Linux, macOS, and FreeBSD.

Library Status

STYDL is currently distributed as part of the AXIS source tree. It can be found in src/vocoder/. A standalone crate release is planned for the future.
