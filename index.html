Command Line Reference

AXIS follows the standard UTAU resampler interface. It is designed to be called by a host application, but can be used manually for testing synthesis quality.

Basic Usage

./axis [input] [output] [pitch] [velocity] [flags] [offset] [length]

Supported Flags (v0.1.0)

Flag Description Range
g Gender: Shifts formants to simulate masculine or feminine tones. g-100 to g100
B Breathiness: Adjusts the noise floor of the WORLD synthesis. B0 to B100

Technical Implementation

AXIS utilizes the WORLD vocoder via the rsworld crate. This allows for high-fidelity extraction of F0, spectral envelopes, and aperiodicity, which are then reconstructed at the target pitch with high precision.
