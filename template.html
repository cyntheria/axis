Standard template for classic software portals.

This layout is inspired by early 2000s software distribution sites. It uses a fixed-width table structure for maximum "retro" compatibility and a maroon-and-white color scheme.

How to use this template

Simply copy the index.html and style.css files to your project. Replace the placeholders with your own content and graphics.

$ Copy layout.html to index.html
$ Edit style.css for custom colors
